### 使用方法

<br>

#### 直接嵌入 iframe

<br>

```html
<iframe
  class="iframe_css_may"
  src="http://d1wpdgybhiu092.cloudfront.net/nftscan/"
  frameborder="0"
></iframe>
```

#### code 演示地址

<br>

[演示地址](http://d1wpdgybhiu092.cloudfront.net/demo.html)

#### 你也可以直接拼接您的钱包地址在我们的链接后面 这样我们就会直接帮你生成对应的元宇宙空间

<br>

```html
<iframe
  class="iframe_css_may"
  src="https://web3.may.social/{您的钱包地址}"
  frameborder="0"
></iframe>
```
